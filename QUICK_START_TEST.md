## 第三步：测试功能

### 方法一：发布页面测试

1. **访问发布页面**
   ```
   http://localhost:5173/publish
   ```

2. **找到上传区域**
   - 看到"添加图片"按钮
   - 旁边应该有"网络图片"按钮

3. **测试网络图片功能**
   - 点击"网络图片"按钮
   - 弹出模态框

     ```
     https://picsum.photos/400/300
     https://picsum.photos/400/301
     https://picsum.photos/400/302
     ```
   - 点击"确认添加"
   - 查看是否添加到上传队列

### 方法二：管理后台测试

如果有管理后台权限，可以在后台的图文管理中测试。

## 🔍 常见问题排查

### 问题 1：看不到"网络图片"按钮

**可能原因**：
- 已达到最大图片数量限制（9 张）
- 组件未正确加载

**解决方法**：
1. 确保当前图片数量 < 9
2. 刷新页面
3. 检查浏览器控制台是否有错误

### 问题 2：点击按钮没反应

**可能原因**：
- JavaScript 错误
- 组件未正确导入

**解决方法**：
1. 打开浏览器控制台（F12）
2. 查看是否有错误信息
3. 检查 `NetworkImageModal.vue` 是否正确导入

### 问题 3：图标不显示

**可能原因**：
- SVG 文件未正确加载

**解决方法**：
1. 检查 `src/assets/icons/link.svg` 是否存在
2. 重新运行 `npm run dev`
3. 清除浏览器缓存

### 问题 4：样式不对

**可能原因**：
- CSS 未生效
- 样式冲突

**解决方法**：
1. 硬刷新页面（Ctrl + Shift + R）
2. 检查浏览器开发者工具中的样式
3. 确认 `.network-upload-item` 样式存在
