## 🔧 故障排查

### 问题 1: 无法访问前端
**检查**: 
- 确认前端服务是否运行（端口 5173）
- 检查浏览器控制台是否有错误

**解决**:
```bash
cd vue3-project
npm run dev
```

### 问题 2: 后端 API 请求失败
**检查**:
- 确认后端服务是否运行（端口 3001）
- 检查数据库连接是否正常

**解决**:
```bash
cd express-project
npm run dev
```

### 问题 3: 数据库连接失败
**检查**:
- MySQL 服务是否启动
- .env 文件中的数据库密码是否正确

**解决**:
```bash
# 更新密码后重启后端
cd express-project
npm run dev
```

### 问题 4: 测试数据丢失
**解决**:
```bash
cd express-project
npm run generate-data
```

## 📝 注意事项

1. **密码安全**: 所有测试账户密码都是简单的 `123456`，仅供测试使用
2. **数据重置**: 每次运行 `generate-data` 会清空现有数据并重新生成
3. **会话过期**: JWT Token 有效期为 7 天，过期需重新登录
4. **刷新令牌**: Refresh Token 有效期为 30 天
5. **自动解封**: 系统每 1 分钟检查一次封禁到期用户
6. **邮件验证**: 当前未启用邮箱验证功能
7. **上传限制**: 文件上传最大 50MB
8. **CORS 配置**: 仅允许 localhost:5173 访问 API
